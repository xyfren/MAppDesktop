# MAppDesktop

Проект состоит из **двух частей**:
- **MAppServer** — серверное приложение, создающее виртуальный (второй) монитор.
- **MAppDriver** — драйвер виртуального дисплея, с которым сервер взаимодействует через **ioctl**.

Сервер используется как часть связки с проектом **MApp** — он формирует второй монитор и передает/принимает данные от клиента.

## Состав репозитория

- `MAppServer/` — серверное приложение (C++/WinAPI).
- `MAppDriver/` — драйвер виртуального дисплея (WDDM/IddSample).
- `MAppDesktop.sln` — решение Visual Studio.

## Требования

- Windows 10/11
- Visual Studio (рекомендуется 2019/2022)
- Права администратора для установки драйвера

## Инструкция по установке и запуску MAppDesktop

### 1. Включение тестового режима (Test Mode)
Так как драйвер может не иметь цифровой подписи, перед установкой необходимо разрешить Windows работу с тестовыми сертификатами:

1.  Откройте **Командную строку** (или PowerShell) **от имени администратора**.
2.  Выполните команду:
    ```bash
    bcdedit /set testsigning on
    ```
3.  **Перезагрузите ПК.** В углу экрана должна появиться надпись «Тестовый режим».

> **Примечание:** Чтобы позже отключить этот режим, выполните `bcdedit /set testsigning off` и перезагрузитесь.

---

### 2. Подготовка файлов
1. Скачайте последнюю версию со страницы **Releases**.
2. Распакуйте архив в удобное место (желательно, чтобы в пути к папке не было русских букв).
3. В папке вы найдете:
    * Папку с драйвером (`.inf`).
    * Установочный файл `INSTALL_DRIVER.bat` (или аналогичный).
    * Приложение `MAppServer.exe`.

---

### 3. Установка драйвера
Драйвер нужен для того, чтобы Windows «увидела» новый виртуальный монитор.

1. Зайдите в папку с распакованными файлами.
2. Нажмите правой кнопкой мыши на файл **`INSTALL_DRIVER.bat`** и выберите **«Запуск от имени администратора»**.
3. Если появится синее окно Windows SmartScreen, нажмите *Подробнее* -> *Выполнить в любом случае*.
4. В процессе установки может появиться запрос на подтверждение установки драйвера без цифровой подписи — нажмите **«Все равно установить этот драйвер»**.
5. После завершения (когда в консоли появится надпись об успехе) **перезагрузите компьютер**.

---

### 4. Запуск и использование
Когда компьютер перезагрузится и драйвер будет готов:

1. **Запуск:** Запустите файл **`MAppServer.exe`**.
2. **Использование:** Теперь вы можете запускать основное приложение **MApp** — оно автоматически свяжется с сервером и начнет трансляцию или взаимодействие.


## Сборка

1. Откройте `MAppDesktop.sln` в Visual Studio.
2. Соберите проекты `MAppDriver` и `MAppServer`.
3. Убедитесь, что драйвер собран (`*.inf`).

Я объединил ваши инструкции в единый логичный процесс: от подготовки системы до первого запуска. Текст стал более структурированным, чтобы пользователь не перескакивал между разделами.

## Связь с MApp

Сервер предназначен для работы в связке с проектом **MApp** и формирует второй монитор на ПК, с которым взаимодействует клиент.
